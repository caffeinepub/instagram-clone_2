# Specification

## Summary
**Goal:** Build a social media application with Instagram-like features including photo/video feed, user profiles, post interactions, stories, direct messaging, and content discovery.

**Planned changes:**
- Create a scrollable feed showing posts from followed users with infinite scroll
- Implement user profile pages with bio, follower counts, and post grid
- Add photo/video upload with caption input and basic image filters
- Enable like, comment, and share functionality on posts
- Implement follow/unfollow system between users
- Create Stories feature with 24-hour expiration
- Build direct messaging system for user-to-user text communication
- Add Explore page for discovering new content and users
- Design modern, photo-focused UI with clean layouts and monochromatic color scheme

**User-visible outcome:** Users can create accounts, post photos/videos with captions, follow other users, interact with posts through likes and comments, share temporary stories, send direct messages, and discover new content through an explore feed.
